AutoCraft

Manage Your Dealership, Drive Your Success!

A automobile dealership management app that allows dealerships to handle auto inventory, customers, sales, and service appointmentsA automobile dealership management app that allows dealerships to handle auto inventory, customers, sales, and service appointments

-Developed a dynamic and interactive user interface using React, leveraging components, props, and state management for efficient and reusable code
-Implemented React Router to enable multi-page navigation within the application, ensuring smooth transitions and a seamless user experience
-Configured Docker to run each microservice on its own container, ensuring scalability and isolation
-Implemented React Hooks to leverage functional components in React and enhance component rendering efficiency

This video demonstrates the vast functionality of AutoCraft.
The navbar provides quick access to the sales, services, and inventory departments.
Dealerships can access salesperson's information and sales history via the sales department.
Creating and managing sales with new or existing customers can be handled in the sales department as well.

This video demonstrates the functionality of services and inventory departments.
All service appointments can be managed in the service department. This includes scheduling new service appointments, viewing appointments and cancelling/completing them, as well as viewing service history
Getting new vehicles in the inventory? You can also manage your automobile inventory in the inventory department! You can add new models, makes that your dealership handles and then add automobiles to be sold!
